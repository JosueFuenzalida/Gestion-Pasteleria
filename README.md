# Gestion Pasteleria

App web HTML/CSS/JS para:

- Agenda de entregas
- Nuevo pedido con selector de productos
- Catalogo y recetas
- Consolidado de compras
- Base de clientes
- Ajustes + backup JSON

## Ejecutar

Abre `index.html` en el navegador.

## Respaldo

En Ajustes puedes exportar/importar un backup unico con estructura:

- `productos.json`
- `pedidos.json`
- `clientes.json`
- `materiales_maestro.json`
- `config.json`
